README for the Whiteboard
=========================

This is a client/server program that let's you draw on a whiteboard with other people.

Pre-requisites:
===============
Java Runtime Environment 8 with JavaFX

How to run:
===========
First you need to run the server in the machine you want to run your Whiteboard clients.
Run this command in the directory where you have the jars installed:
java -jar Whiteboard-Server.jar
Then you can run each of the clients you want on your machine using this command (in the same directory):
java -jar Whiteboard-Client.jar
